# QuickUstcWlt
一个用 Qt 写的快捷登陆 USTC 网络通的 GUI 应用，支持 Windows 与 macOS

* 系统托盘可直接查看当前网络状态（角标数字即为通道）

* 可设置定时查询与定时登陆功能，时刻联网

P.S. 用 Qt 6.2.4 写的，windeployqt / macdeployqt 直接打包，未精简依赖，封装出来文件比较大，实际工程很小
